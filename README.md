# odin-recipes
TOP HTML Project - Build a Basic Recipe Website

This project is to take what we've learned about HTML thus far and applying it to make a basic recipes website. 

<h2>Objectives/Goals:<h2>

<ul>
    <li>Use the different types of ways to edit text (bold, italicized, and headings)</li>
    <li>Create both a standard list and an ordered list</li>
    <li>Implement absolute and relative links</li>
    <li>Insert images into the website (use at least 2 of the 4 file types)</li>
    <li>Get familiar with writing good commit messages</li>
</ul>
