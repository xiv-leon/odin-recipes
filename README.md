# odin-recipes
TOP HTML Project - Build a Basic Recipe Website

This project is to take what we've learned about HTML thus far and applying it to make a basic recipes website. 

Objectives/Goals:

*Use the different types of ways to edit text (bold, italicized, and headings)
*Create both a standard list and an ordered list
*Implement absolute and relative links
*Insert images into the website (use at least 2 of the 4 file types)
*Get familiar with commit messages
